重点知识

OSX 和 iOS 中的共享库在/usr/lib 中。

Mach-O 替代了其他所有 UN*X 系统的 ELF 可执行文件格式。

类型 Mach-O 动态库 ELF
动态库 后缀 .dylib .so
libc /usr/lib/libc.dylib -> libSystem.B.dylib libc.so
libc 的功能 数学函数、主机名解析、线程库等功能 -lm、-lnsl、-lpthread等
核心库 libSystem 提供了 系统调用、内核服务接口、保护/usr/lib/system 下的一些库

Mach-O 中 __stubs(桩区), 这个区用来存放本地未定义的符号的占位符。
otool -l xxx | grep sectname

编译器会创建对 __stubs 区中符号的调用，连接器在运行时解析对桩的调用。
链接器实现跳转的原理是在 xx 处放置一条 JMP 指令。

相关工具

  1. otool -L : 显示库的依赖关系、
  2. otool -p func_name -tV xxx.dyld : 查看汇编代码

  3. nm -m : 显示二进程文件的符号表，-m 参数还能跟踪符号的解析过程

  4. dyldinfo : xcode 中的工具，和 nm 差不多。
